# Project Instructions

このプロジェクトは、Web制作練習用の架空カフェLPです。

## 方針
- まずREADMEに仕様を整理してから実装する
- HTML/CSS/JavaScriptのみで作る
- ReactやNext.jsは使わない
- 外部APIは使わない
- 画像は外部サービスに依存せず、CSSの背景やプレースホルダーで表現する
- スマホ表示を重視する
- レスポンシブ対応を必ず行う
- アクセシビリティを意識する
- 個人情報は載せない
- 実装後にREADMEへ表示確認方法を追記する

## 技術方針
- HTMLは意味のあるタグを使う
- CSSは読みやすく整理する
- JavaScriptは必要最小限にする
- スマホメニューや簡単なUI操作のみ実装する
